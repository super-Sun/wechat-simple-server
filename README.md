# wechat-signature-server
(nodejs)微信 JS-SDK 签名
针对微信浏览器开发者，提供获取 access_token/jsapi_ticket/signature 等接口。

请自行配置 `src/config.js` 中的 `appId` 和 `appSecret` 以及公众号安全域名。

## 功能列表
+ 获取 access_token
+ 获取 jsapi_ticket
+ 获取 signature

## install
```bash
npm install
```

## usage
```bash
npm run start
```
